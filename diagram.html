RAG Chatbot — User Query Flow

Frontend Browser
User types a question → script.js sends POST /api/query
  • Payload: { query, session_id }
  • Displays answer + source citations on response
HTTP POST /api/query
app.py FastAPI
  • Create session if none exists → SessionManager
  • Delegate to rag_system.query(query, session_id)
  • Return { answer, sources, session_id }
rag_system.py Orchestrator
  • Load conversation history from SessionManager
  • Forward query + history + tool definitions → AIGenerator
  • Collect sources from ToolManager after generation
  • Save exchange back to SessionManager
ai_generator.py
Claude API
  • Sends query + history to Claude
  • Claude decides to call CourseSearchTool
  • Receives chunks, generates grounded answer
tool call / chunks
vector_store.py
ChromaDB
  • Embed query with Sentence Transformers
  • Cosine similarity search
  • Return top-N document chunks
answer + sources
Frontend Browser
Renders answer with cited source documents
